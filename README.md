## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Installation of latest version of NPM
* npm
  ```sh
  npm install npm@latest -g
  ```



### Installation

1. Install NPM packages
   ```sh
   npm install -g hello-name-cli
   ```
